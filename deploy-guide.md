# 오늘 뭐 먹지 — 배포 가이드 (v1.1 · 실시간 데이터)

이 폴더(`todayeat-pwa`)가 앱 전체입니다.

```
todayeat-pwa/
├─ index.html              ← 앱 본체
├─ manifest.webmanifest    ← 앱 정보(이름·색·아이콘)
├─ service-worker.js       ← 오프라인 캐시
├─ icon-192.png / icon-512.png  ← 앱 아이콘
├─ api/
│   └─ nearby.js           ← 주변 식당 조회 서버(카카오 프록시)  ★신규
└─ deploy-guide.md            ← 이 문서
```

---

## 배포 방식은 두 가지

| 방식 | 실시간 주변 식당 | 난이도 | 추천 |
|---|---|---|---|
| **A. Netlify Drop (정적)** | ❌ 강남역 샘플만 | 아주 쉬움 | 빠른 공유·UI 확인용 |
| **B. Vercel (서버 포함)** | ✅ 내 위치 실시간 | 보통 | 실제 사용용 (권장) |

> 앱은 두 방식 모두에서 켜집니다. 차이는 "실시간 위치 기반 식당"이 되느냐뿐이에요. 위치 권한을 허용해도 서버(api)가 없으면 자동으로 강남역 샘플로 보여줍니다.

---

## A. 빠르게 확인만 — Netlify Drop
1. **https://app.netlify.com/drop** 접속
2. `todayeat-pwa` 폴더를 통째로 드래그&드롭
3. 몇 초 뒤 `https://…netlify.app` 주소 생성 → 공유

이 방식은 `api/` 서버가 동작하지 않아 **강남역 샘플 데이터**로만 보입니다. UI·설치·투표·기록을 확인하기엔 충분해요.

---

## B. 실시간 데이터까지 — Vercel (권장)

### 1) 카카오 REST API 키 발급 (무료)
1. **https://developers.kakao.com** 로그인 → "내 애플리케이션" → **애플리케이션 추가하기**
2. 만든 앱 → **앱 키** 메뉴 → **REST API 키** 복사 (이게 서버에 넣을 키)
3. 좌측 **카카오맵** 메뉴에서 사용 설정을 켭니다 (로컬 API 사용).

> ⚠ 여기서 복사하는 건 **REST API 키**입니다. JavaScript 키가 아니에요.

### 2) Vercel에 배포
1. **https://vercel.com** 가입(깃허브/구글 로그인)
2. **Add New → Project** → `todayeat-pwa` 폴더 업로드(또는 GitHub 저장소 연결)
3. 별도 빌드 설정 없이 그대로 Deploy — `api/nearby.js`는 Vercel이 자동으로 서버 함수로 인식합니다.

### 3) 환경변수에 키 넣기 (가장 중요)
1. Vercel 프로젝트 → **Settings → Environment Variables**
2. 이름 `KAKAO_REST_KEY`, 값에 1)에서 복사한 **REST API 키** 입력 → 저장
3. **Deployments → 최근 배포 → Redeploy** (환경변수는 재배포해야 적용됨)

이제 배포 주소를 폰에서 열고 **위치 권한을 허용**하면, 상단에 "현재 위치 · 실시간 주변"으로 바뀌고 내 주변 식당이 채워집니다.

- **안드로이드(크롬)**: 주소 접속 → "앱 설치" 배너/메뉴

## 지금 되는 것 / 아직 남은 것

**됨**
- 내 위치(GPS) 기반 실시간 주변 식당 (카카오 로컬) · 시간대별 추천
- 카테고리 필터 · 겹친 카드 UI · 먹은 기록/별점/선호 · 익명 투표 · 홈 설치 · 오프라인
- 카카오맵 위치(지도 버튼) · 커피 후원(카카오페이)

**아직 (다음 단계 후보)**
- **사진**: 카카오 검색은 업체 사진을 안 주기 때문에, 지금은 카테고리별 대표 사진을 씁니다. 실제 업체 사진은 카카오/네이버 플레이스 상세 연동 시 교체.
- **메뉴/가격**: 카카오는 상호·카테고리까지만 줍니다. 세부 메뉴·가격은 플레이스 상세나 제휴 단계에서 보강.
- **배달 딥링크 · 사용자 계정(백업/동기화) · 실시간 공유 투표 · 푸시**.

## 참고 — API 한도
- 카카오 로컬: 앱당 일 30만 호출(무료). 서버가 좌표 반올림 기준 5분 캐시해 호출을 아낍니다.
- 키는 서버(환경변수)에만 있고 앱(브라우저)에는 노출되지 않습니다.

이어서 개발할 부분이 있으면 클로드에게 "사진 연동하자", "배달 딥링크 붙이자" 처럼 말씀하시면 됩니다.
